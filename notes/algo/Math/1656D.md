# 1656D Codeforces

## Problem 
Given $n$, find $k$ such that: $n = \sum_{i=0}^{k-1} a_i$ and $ a_i \equiv i \pmod{k}$

[Link](https://codeforces.com/contest/1656/problem/D)

## Approach 
***Observations***. 
1. $n$ has the form: $ \frac{k(k+1)}{2} + km$.
2. If $n \equiv 1 \pmod{2}$, choose $ k=2 $
From 1. we can choose $k$ to be an odd divisor of $n$ if $\frac{k(k+1)}{2} \le n$.
Otherwise, choose $k=2^{x+1}$ where $n = 2^{x}d$.
It is obvious that this works for the first case, so let's check the second case. 
Set $k = 2^{x+1}$ in formula (1), we have: 
$2^{x}d = (2^{x+1}+1)2^{x} + 2^{x+1}m $
$\Leftrightarrow d = (2^{x+1} + 1) + 2m$  
$\Leftrightarrow d \equiv 1 \pmod{2}$ (always true)
## Solution

For each test case, check whether n is odd or even. 
* If n is odd, the answer is 2 
* If n is even, find the odd divisor d such that: $ n = 2^{x}d $.
    + If $d = 1$, the answer is -1
    + If $\frac{d (d+1)}{2} \le n$, the answer is d 
    + Otherwise, the answer is $2^{x+1}$
