# [1811 E](https://codeforces.com/problemset/problem/1811/E) 

`Math`

## Problem 
A sequence a contains all natural numbers that do not contain the digit 4. 
Find $a_k$

## Solution 
Just convert from base-10 to base-9. 
Notice that if digit x in base-9 > 4 the we should print x+1
